# Direccion visual

## Base

Tomar la calidez, el caracter editorial y el pulso creativo de Paisanos. El
lenguaje de viaje funciona como metafora secundaria del Paisaporte, no como
estetica aerolinea corporativa.

## Lenguaje de interfaz

- Paisaporte: credencial central del miembro.
- Encuentro: unidad operativa de la comunidad.
- Punto: lugar/token de entrada a un evento.
- Conexiones: sugerencias de networking.
- Sellos: hitos futuros del Paisaporte, no niveles de miembros.
- Torre de control: admin panel.

## UI

- Bordes de 2-4px de radio, no cards excesivamente blandas.
- Superficies tipo papel, tinta oscura, acentos vivos y una expresion mas Paisanos que aerolinea.
- Iconos lucide para acciones y estados.
- Densidad media: mas producto que marketing.
- Mobile-first para evento y QR.

## Primer prototipo

La pantalla inicial debe modelar:

- Experiencia editorial de producto
- Paisaporte como identidad viva
- Evento proximo
- Cruces sugeridos
- Radar de oportunidades
- Continuidad antes/durante/despues del evento
